# 👕 Clothing Website

A stylish and responsive **Clothing Website** built using HTML, CSS, and JavaScript.  
This project showcases fashion products with an attractive UI and smooth browsing experience.

---

## 🚀 Features

- 🏠 Home page with hero section
- 🛍️ Shop page with product listings
- 📄 Single product detail page
- 🎨 Modern UI with banners and images
- 📱 Fully responsive design
- ⚡ Interactive elements using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
